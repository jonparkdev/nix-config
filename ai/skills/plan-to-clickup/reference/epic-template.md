# Epic Template

Use this template for ClickUp epic task descriptions. Fill in each field from the decomposed plan data.

```
**Epic Name:** [Epic Title or ID]

**Objective:** A concise statement describing the outcome the epic aims to achieve.

**Business Value:** An explanation of how the business can expect to gain value from this work.

**Scope:** A brief description of what is included in the epic.

**Background:** Key information or data justifying the epic.

**Stakeholders:** List of departments or roles impacted.

**Dependencies:** Critical dependencies that affect this epic.

**Acceptance Criteria:**

- Functional Requirement 1: Description
- Functional Requirement 2: Description
- Non-Functional Requirement 1: Description
- Compliance Requirement: Specific standard or regulation

**Milestones:** (Optional) Key dates or phases for tracking progress. These may be points in time related only to the epic in focus, or they may be external events that interrelate.

**TDD:** A link to the TDD(s) relevant to this epic or an explanation of why this epic does not need one.

**Risks:** Short description of major risks and mitigation strategies.
```
